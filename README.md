# Team Emailing Management Dashboard

A professional, high-end management tool for emailing teams to track RPs (Domains) and Servers.

## Features
- **Role-Based Access**: Secure login for Admin and Mailers.
- **Resource Management**: Drag-and-drop interface for assigning RPs and Servers to team members.
- **Live Overview**: Real-time summary of team activity and resource allocation.
- **Premium UI**: Dark-themed design with glassmorphism and modern aesthetics.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6 Modules)
- Lucide Icons

## Deployment
This project is ready for zero-config deployment on **Vercel**. Just push to your repository and Vercel will automatically detect the static site.

## Setup
No build step required. To run locally:
1. Clone the repository.
2. Open `index.html` in your browser or run a simple server:
   ```bash
   python -m http.server
   ```
